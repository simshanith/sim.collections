Namespace Detail
calitha.collections
The collections namespace consists of static methods that supports the use of the collections framework. The methods operate or return collections or comparators. Probably the most important methods are those that make integer-based numbers and strings compatible as keys in hash-based collections.

<static> calitha.collections.makeNumberHashCompatible(number)
Makes a number with integer value compatible with hash-based collections. This is done by extending the object to include an equals and hashCode value. You can pass the Number.prototype to make all existing and new numbers compatible.
